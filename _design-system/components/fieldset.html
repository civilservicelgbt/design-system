Use the fieldset component to group related form inputs.

When to use this component

Use the fieldset component when you need to show a relationship between multiple form inputs.

Fieldset with extra large legend

Formatted example
Fieldset legend <!-- form fields go here -->
HTML code 
<fieldset id="fieldset-group-name" aria-describedby="fieldset-group-title">
    <legend id="fieldset-group-title" class="legend-xl">
        Fieldset legend
    </legend>
    <!-- form fields go here -->
</fieldset>

Field set with medium legend

If you do not add the legend-* class to the <legend> tag, it will default to medium size.

HTML code 
<fieldset id="fieldset-group-name" aria-describedby="fieldset-group-title">
    <legend id="fieldset-group-title" class="legend-md">
    Fieldset legend
    </legend>
    <!-- form fields go here -->
</fieldset>
